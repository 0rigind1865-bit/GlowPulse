# 參考貼文庫

收錄值得學習的優質 Threads 貼文，作為 AI 風格分析的學習素材。
URL 僅供追溯來源，分析時只使用「文字」欄位的內容。

---

## 使用說明

1. 在 Threads 看到好貼文 → 複製網址與文字貼到對應分類
2. 填寫「為何收錄」：說明這篇的亮點（切入角度、句型、情緒）
3. 執行分析指令（待開發）後，AI 會從這份文件學習寫作模式

---

## 新增格式

每筆參考貼文請依以下格式填寫：

```
#### 貼文編號（流水號）
URL: https://www.threads.net/@帳號/post/ID
文字：
（貼上完整貼文原文，保留原始換行）
為何收錄：（說明亮點，20 字以內）
```
